Introduction to ConvNets: Classifying handwritten numbers

Let's take a look at a simple example of a convnet. We will use it to classify the MNIST dataset, which is an open dataset containing handwritten numbers.

Let's create a first basic convnet. It's a stack of 'Conv2D' and 'MaxPooling2D' layers. The important thing to note is that a convnet takes as input tensors of size (image_height, image_width, image_channels). To do this we must first find out the size of the images in our dataset.

The network must have the following layers:

  • A convolutional layer (Conv2D) with 32 3x3 filters and relu activation. In this first layer you must indicate the size of the input (input_shape).
  • A second layer of Max Pooling (MaxPooling2D) of 2x2
  • A third convolutional layer with 64 3x3 filters and relu activation
  • A fourth layer of 2x2 Max Pooling (MaxPooling2D)
  • A fifth convolutional layer of 64 3x3 filters and relu activation

You'll know you've done it right when the model.summary() output is:

Con base en las indiaciones dadas para cada capa, se crea la red. La primera capa aprende 32 filtros con una ventana tamao 3x3, donde cada filtro se desliza (convolution) sobre la imagen de entrada, produciendo 32 mapas de activacin. La activacin ReLU (Rectified Linear Unit) se utiliza para aadir no linealidad y ayuda a la red a capturar patrones ms complejos. La segunda capa realiza un pooling de 22, reduciendo la dimensin espacial de los mapas de activacin; donde, al reducir la resolucin espacial hace que la red se concentre en caractersticas ms globales y se reducen los parmetros, evitando sobreajuste y bajando el coste computacional. Luego se aplica otra convolucin sobre la salida reducida anterior con 64 filtros 3x3 y la funcin de activacin ReLu, para aprender caractersticas ms complejas. En una cuarta capa se reduce nuevamente la dimensin especial 2x2; y finalmente, una tercera convolucin con 64 filtros de 3x3, donde en esta fase la red ya se encuentra extrayendo rasgos de mayor nivel, combinando la informacin de las capas anteriores.

# Definicin del modelo
model = models.Sequential()

#Capa Conv2D con 32 filtros 3x3, activacin ReLU y forma de entrada (28, 28, 1)
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))
# Capa de MaxPooling2D 2x2
model.add(layers.MaxPooling2D((2, 2)))
#Capa Conv2D con 64 filtros 3x3 y activacin ReLU
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
#Capa de MaxPooling2D 2x2
model.add(layers.MaxPooling2D((2, 2)))
# Capa Conv2D con 64 filtros 3x3, activacin ReLU
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

#salida
model.summary()

You can see above that the output of each Conv2D and MaxPooling2D layer is a 3D tensor of dimensions (height, width, channels). The width and height tend to decrease as we go deeper into the network. The number of channels is controlled by the first argument passed to the Conv2D layers (e.g. 32 or 64).

The next step would be to give our last tensor (of dimensions (3, 3, 64)) as input to a densely connected network. These classifiers process vectors, which are 1D, while our output is a 3D tensor. So first we will have to flatten our 3D output and convert it to 1D and then add a few dense layers:

  • Add a first layer of 64 neurons
  • Add a last layer of 10 neurons (as many as you can sort) and softmax activation
  • You'll know you've done well when the summary looks like this:

Despus de la ltima capa convolucional se obtiene un tensor 3D de dimensiones (3, 3, 64), donde se contienen caractersticas de alto nivel de la imagen. Sin embargo, para poder generar una salida que clasifique los nmeros hay que conectarlo a una capa densa que opera sobre vectores 1D, por lo cual se debe convertir el volumen 3D que sale de la ltima convolucin en un vector 1D. La salida (3, 3, 64) tiene un total de 3 * 3 * 64 = 576 valores, los cuales son aplanados con la funcin Flatten. Seguidamente, es posible conectar la capa densa con 64 neuronas y la funcin de activacin ReLu, para luego aadir la capa densa con las 10 neuronas de salida para cada nmero, con una funcin de activacin Softmax. Esta capa es el clasificador que reciba las caractersticas extradas y decida a qu clase pertenece la imagen. Esta ltima capa cuenta con una funcin de activacin diferente debido a que cada neurona de esta capa corresponder a la probabilidad estimada de que la imagen pertenezca a la clase x (valores de 0 a 9), y esta funcin es usada para problemas de clasificacin mltiple ya que fuerza a que la salida sea un vector de probabilidades que sumen 1.
