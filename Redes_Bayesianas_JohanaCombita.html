Cul sera el nmero potencial de parmetros (sin usar la red bayesiana) del modelo para calcular la probabilidad global si no utilizasemos el DAG?

Para un conjunto de n variables binarias, si no se usa ninguna factoracin, es decir, se modela la distribucin conjunta de forma plana, el nmero de parmetros necesarios para describir la probabilidad de cada combinacin sera 2^n 1. Esto quiere decir que hay 2^n celdas en la tabla completa y la suma de estas debe ser 1. Cmo en este ejercicio se tienen 10 variables, entonces:

Los resultados muestran la reduccin significativa de parmetros para el modelo, al usar una red bayesiana. Esto se debe a que la red bayesiana impone independencias condicionales, permitiendo descomponer la distribucin conjunta en factores locales ms pequeos; lo cual genera pasar de 1023 parmetros (para el caso de 10 variables binarias) a 21.

La clave de la reduccin de parmetros est en que cada variable se modela solo con sus padres. Esto impone independencias condicionales que evitan que se tenga que estimar todas las combinaciones posibles de las dems variables.

Es importante mencionar que este permite un menor sobreajuste (menos parmetros), mayor interpretabilidad (mejor lectura de dependencias) y calculos eficientes (inferencias ms manejables).

Ahora bien, el nmero de parmetros asociado a cada una de las distribuciones locales es:

Para cada nodo binario dentro de la red, solo necesitamos un nmero de parmetros igual al nmero de combinaciones posibles de sus padres. Si un nodo tiene k padres, hay 2^k maneras distintas de asignarles valores, debido a que cada padre puede tomar dos valores, 0 o 1. Para cada una de esas combinaciones, se requiere un nico parmetro que indique la probabilidad de que el nodo sea 1, y la probabilidad de que sea 0 se obtiene restando ese valor de 1. Por tanto, el total de parmetros de la red se calcula sumando para cada nodo la cantidad de combinaciones de sus padres.
